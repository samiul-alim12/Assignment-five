Question-1: What are the different ways to select an element in the DOM?

in DOM I can select an element by :
  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

difference between innerText, innerHTML and textContent :

innerHTML : innerHTML property used to get access or modify the content of an element

innerText : innerText property used to get access or modify the visible text content of an element.

textContent: text content property used to get the text content of an element, ignoring any html tags.

Question-3: What is event delegation in the DOM?

event delegation ব্যবহার করে শুধুমাত্র প্যারেন্ট এলিমেন্ট ইভেন্ট লিসেনার সংযুক্ত করলে তা পুরো চাইল্ড এলিমেন্টের ইভেন্টগুলো ক্যাপচার করতে পারে

Question-4: What is event bubbling in the DOM?

এটি এমন একটি প্রক্রিয়া যেখানে একটি ইভেন্ট প্রথমে তার টার্গেট এলিমেন্টে ঘটবে । এবং তারপর প্যারেন্ট এলিমেন্ট গুলোর দিকে propagate করে ।
