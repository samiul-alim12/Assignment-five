Question-1: What are the different ways to select an element in the DOM?

in DOM I can select an element by :
  1. getElementById()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()
  5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

difference between innerText, innerHTML and textContent :

innerHTML : innerHTML প্রপার্টি ব্যবহৃত হয় প্রধানত HTML কন্টেন্ট অ্যাক্সেস এবং পরিবর্তন করার জন্য। এটি সাধারণত ব্যবহার করা হয় যখন একটি HTML উপাদানের ভিতরের কন্টেন্ট (যেমন টেক্সট, ট্যাগ, অথবা অন্য কোন HTML উপাদান) নিতে বা সেট করতে হয়।

innerText : innerText ব্যবহার করে একটি এলিমেন্টের শুধুমাত্র দৃশ্যমান টেক্সট অ্যাক্সেস করা যায়। innerText ব্যবহার করে একটি এলিমেন্টের ভিতরের টেক্সট ও পরিবর্তন করা যায়.

textContent: textContent ব্যবহার করে একটি এলিমেন্টের সম্পূর্ণ টেক্সট পাওয়া যায় । এমনকি যদি সেটি সিএসএস দিয়ে লুকানো থাকে যেমন (display: none বা visibility: hidden) তারপরও সেটি পাওয়া যায় । এটি এইচটিএমএল ট্যাগ উপেক্ষা করে শুধুমাত্র টেক্সট রিটার্ন করে

Question-3: What is event delegation in the DOM?

event delegation ব্যবহার করে শুধুমাত্র প্যারেন্ট এলিমেন্ট ইভেন্ট লিসেনার সংযুক্ত করলে তা পুরো চাইল্ড এলিমেন্টের ইভেন্টগুলো ক্যাপচার করতে পারে

Question-4: What is event bubbling in the DOM?

এটি এমন একটি প্রক্রিয়া যেখানে একটি ইভেন্ট প্রথমে তার টার্গেট এলিমেন্টে ঘটবে । এবং তারপর প্যারেন্ট এলিমেন্ট গুলোর দিকে propagate করে ।
